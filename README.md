# Smart Elephant Fence Breach Detector System

## Overview
The Smart Elephant Fence Breach Detector System is a real-time IoT-based monitoring system designed to detect fence breaches caused by elephants and trigger instant alerts through mobile and communication networks.

The system integrates embedded hardware, GSM communication, and a mobile application to improve early warning and response in human–elephant conflict areas in Sri Lanka.

---

## Technologies Used

### Mobile Application
- Android Studio
- Java / XML
- Firebase Authentication
- Firebase Cloud Messaging (FCM)

### Backend & Communication
- REST API
- Firebase Cloud Services
- GSM (SIM800A module)

### Embedded System
- ESP32 Microcontroller
- BMI160 Accelerometer / Gyroscope

### Tools
- Arduino IDE
- Git & GitHub
- Android Studio

---

## Key Features
- Real-time fence breach detection
- Instant alert system via mobile notifications
- GSM-based emergency communication
- Cloud-based backend integration
- Multi-platform system architecture (Embedded + Mobile + Cloud)

---

## Purpose
This project demonstrates an IoT-based real-time alert system combining embedded systems, mobile application development, and cloud integration for environmental and wildlife safety applications.
